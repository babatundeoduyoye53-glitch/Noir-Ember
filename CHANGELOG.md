# Changelog

All notable changes to the Noir & Ember restaurant website will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024

### Added
- Initial release of Noir & Ember restaurant website
- Full responsive design for mobile, tablet, and desktop
- Dark/Light theme switching with persistent preference
- Dynamic menu system with 30 dishes across 6 categories
- Advanced filtering: categories, dietary preferences, and search
- Reservation system with form validation
- Contact page with map integration
- About page with restaurant story and team information
- Image gallery with lightbox functionality
- Customer testimonials carousel
- Chef's specials section with weekly menu and private dining
- FAQ accordion component
- Mobile-optimized filter drawer with bottom sheet animation
- Loading skeletons for better perceived performance
- SEO optimization with meta tags and structured data
- Accessibility features (ARIA labels, keyboard navigation)
- PWA support with manifest.json

### Technical Features
- React 18 with modern hooks
- React Router 7 for client-side routing
- Vite 6 for fast builds and HMR
- Custom CSS with CSS variables for theming
- Lucide React for consistent iconography
- All images optimized and stored locally
- Error boundary for graceful error handling

### Performance
- Lazy loading for images
- Route-based code splitting
- Optimized bundle size
- Smooth animations with CSS transitions
- Mobile-first responsive design
